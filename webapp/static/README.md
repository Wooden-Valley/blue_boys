# Статические файлы

Положите сюда изображение, сгенерированное нейросетью, под именем **`banner.png`** —
оно автоматически появится в шапке главной страницы (`/`). Пока файла нет,
на главной показывается пунктирный слот-заглушка.

Промпты для генерации (Kandinsky / Шедеврум / Midjourney и т.п.):

- **Баннер шапки:** «Уютная корзина со свежими продуктами (овощи, фрукты, молоко,
  хлеб) на светлом фоне, минимализм, мягкий свет, зелёно-белая палитра, широкий
  баннер для сайта доставки продуктов, без текста».
- **Маскот-помощник (по желанию, заменить эмодзи 🥕):** «Дружелюбный мультяшный
  персонаж-морковка с улыбкой, простой флэт-стиль, зелёно-оранжевые цвета,
  на прозрачном фоне».

В презентации покажите использованный промпт и итоговый результат (требование методички).
