# Instacart Client Insights

Учебный проект: интеллектуальный клиентский сервис на данных покупательского
поведения (датасет **Instacart**). Приложение по идентификатору клиента получает
его историю покупок из аналитического сервиса **Loginom**, показывает её в
веб-интерфейсе и формирует персональную рекомендацию с помощью большой языковой
модели **Mistral**.

Стек: **Loginom** (low-code аналитика, REST) → **Flask** (веб-приложение) →
**Mistral** через `langchain` (рекомендации), с кэшированием ответов модели.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate            # Windows
pip install -r webapp/requirements.txt
```

Создайте файл `.env` в корне с ключом Mistral:

```
MISTRAL=ваш_api_ключ
```

Запуск приложения:

```bash
cd webapp
python app.py                    # http://localhost:8000
```

## Тесты

```bash
cd webapp
python -m pytest
```

## Как это работает

1. **Авторизация** — клиент вводит `user_id`; приложение проверяет его наличие
   через Loginom-сервис `GetUserHistory`.
2. **Анализ** — из Loginom приходит топ товаров клиента; данные показываются
   таблицей и столбчатой диаграммой.
3. **Рекомендация** — история покупок передаётся в Mistral, модель возвращает
   персональный текстовый анализ.
4. **Кэширование** — ответ модели сохраняется в `webapp/llm_cache.json` по ключу
   `sha256(user_id + список товаров)`; при повторном запросе с теми же
   параметрами ответ берётся из кэша, а не запрашивается у модели заново.
